# Voici un guide étape par étape pour lancer L'application

1-Assurez-vous que votre machine a Node.js et npm (le gestionnaire de paquets de Node.js) installés. Vous pouvez vérifier cela en exécutant les commandes node --version et npm --version dans un terminal
telecharger node js à l'url  suivant <https://nodejs.org/en/download>

2-Accédez au répertoire racine de votre application React en utilisant la commande cd <nom_du_répertoire> dans un terminal

3-Installez les dépendances de votre application en exécutant la commande npm install dans votre terminal. Assurez-vous que le fichier package.json est présent dans le répertoire racine de l' application.

4-Lancez votre application React en utilisant la commande npm run dev. Cela lancera un serveur de développement et vous pourrez accéder à votre application en ouvrant votre navigateur web et en accédant à l'adresse <http://localhost:5173/>
